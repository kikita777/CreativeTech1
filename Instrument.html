Project 4: Needle & Thread

Techiques & Platforms: Sewing machine, MakeyMakey, MAX

Aim

For this project, my aim is to play around with MakeyMakey and to see how it can be connect with other mundane objects which already in my room. I started with fruits then tried using copper tape, sewing machine, needles to build up the circuit. And I connected the key with noteout in MAX, so that the function of these physical objects were shifted into a simple instrument.
